# Start Here

This book collects the complete Paper II (Upper Primary, Classes VI–VIII, Science/Mathematics stream) from the last five CTET sessions:

- August 2023
- January 2024
- July 2024
- December 2024
- February 2026

Every question is transcribed from the official CBSE question paper PDFs published at ctet.nic.in/archive, and every correct answer is checked against the official final answer key PDF for the matching set code. Nothing here is copied from a coaching site — the questions are official, and the explanations are written fresh for this book.

## How to use this book

Each session gives you two chapters:

1. **Questions** — the full 150-question paper, section by section, with nothing given away. Attempt it like the real exam: no peeking, time yourself if you can (150 questions / 150 minutes).
2. **Answers and Explanations** — the same questions in the same order, each with the correct option and a short explanation of why it's correct (and, where useful, why the near-miss options are wrong).

Work through one session fully — solve, then check — before moving to the next. Sessions are ordered newest first, since the most recent paper is the best signal for current exam pattern; if you'd rather build up chronologically, just read the chapters oldest-session-first instead.
