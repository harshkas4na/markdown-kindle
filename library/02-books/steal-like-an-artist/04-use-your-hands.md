# 4. Use Your Hands

**The idea in one line:** working with physical tools — pen, paper, scissors, a whiteboard — produces different raw material than a screen does, so a purely digital practice is missing a whole mode of thinking.

## The computer is an editing tool, not a generating one

Screens are excellent for refining, arranging, and distributing work, but they're a comparatively poor environment for the messy, exploratory stage where ideas first show up. A cursor blinking in an empty document invites you to write something that already looks finished, because the interface makes even a rough first attempt look polished. Physical materials don't have that problem — a scrawl on paper looks like a scrawl, which makes it easier to treat as disposable, which in turn makes it easier to take risks with.

## Analog friction produces different ideas

Cutting, pasting, sketching, or writing by hand engages you differently than typing does — it's slower, which sounds like a disadvantage but often isn't, because the extra friction forces more deliberate choices and surfaces connections a fast digital workflow skips over. Working across two different media — say, sketching on paper and then digitizing — also means each pass filters the idea differently, so the version that survives both passes tends to be sturdier than one that only ever existed on a screen.

## A simple physical practice is enough

This doesn't require elaborate materials. A notebook for jotting first-draft thoughts before opening a document, index cards for arranging a structure before typing it, a whiteboard for a rough diagram before formalizing it in a tool — any of these reintroduce the physical stage without requiring you to become a different kind of maker. The point isn't the specific tool, it's keeping a non-digital step somewhere upstream of the polished digital output.

## Boredom and thinking need room

Screens are also very good at filling every idle moment, and idle, unfilled moments are frequently where connections between unrelated ideas actually form. A practice that includes time away from a screen — even just the walk to get the notebook — leaves room for the kind of undirected thinking that a constantly-occupied attention span crowds out.

## Takeaways for practice

- Start new ideas on paper or a whiteboard before opening a document or editor.
- Use a notebook for capture; use the computer for arranging and finishing.
- Deliberately schedule unstructured, screen-free time as part of the creative process, not as a break from it.
- If a project has stalled on-screen, take the same problem to paper and see if it moves.
